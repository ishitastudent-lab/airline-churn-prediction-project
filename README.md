# ✈️ Airline Customer Churn Prediction

## 📌 Project Overview

This project predicts whether an airline customer is likely to churn using machine learning techniques. The objective is to identify customers at risk of leaving so that airlines can take proactive retention measures.

The project covers the complete machine learning workflow including data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and prediction.

---

# 🎯 Objective

- Analyze customer behaviour
- Predict customer churn
- Identify important factors affecting churn
- Build a complete end-to-end machine learning pipeline

---

# Dataset

The dataset contains customer-related information such as:

- Customer demographics
- Flight behaviour
- Loyalty status
- Travel history
- Service usage
- Other customer attributes

Target Variable:

- **Churn (0 = No, 1 = Yes)**
